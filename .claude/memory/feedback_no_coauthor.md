type: feedback

Do not add `Co-Authored-By: Claude ...` to commit messages. Claude is a tool, not a person or contributor.

**Why:** Claude is the development tool, not a contributor to the project.

**How to apply:** Simply omit the Co-Authored-By line from all commit messages.
