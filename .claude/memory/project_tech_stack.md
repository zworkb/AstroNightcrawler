The project uses **NiceGUI** as the web framework for the browser-based application.

**Why:** User's deliberate choice — the project directory structure already reflects this decision.

**How to apply:** Design all UI components using NiceGUI's component model. Consider NiceGUI's strengths (Python-only, reactive, built-in elements like charts/tables/interactive widgets) and limitations (less flexible than raw JS frameworks for highly custom UIs) when proposing architecture and UI solutions.
